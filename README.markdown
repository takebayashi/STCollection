# STCollection
STCollection is the useful categories for Cocoa collection classes.

## Features

The following features are supported in v0.1.0:

* Array
    * reverse
    * flatten
    * map
    * filter
    * grouped
    * fold left
    * fold right
    * reduce left
    * reduce right
* Dictionary
    * map
* Index Set
    * all indexes
    * map
