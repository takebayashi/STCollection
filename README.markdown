# STCollection
STCollection is the useful categories for Cocoa collection classes.

## Features

The following features are supported in v1.0.0:

* Array
    * reverse
    * flatten
    * map
    * filter
    * grouped
    * zip
    * unzip
    * fold left
    * fold right
    * reduce left
    * reduce right
    * etc.
* Dictionary
    * map
    * filter
    * etc.
* Index Set
    * map
    * etc.
